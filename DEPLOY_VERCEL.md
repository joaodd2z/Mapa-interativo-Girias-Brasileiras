# 🚀 Como fazer deploy no Vercel

## Deploy rápido

### 🔥 **Opção 1: Via GitHub (Recomendado)**

```bash
# 1. Subir para GitHub
git init
git add .
git commit -m "🇧🇷 Mapa Interativo de Gírias Brasileiras"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/mapa-girias-brasileiras.git
git push -u origin main

# 2. Deploy no Vercel
# - Acesse vercel.com
# - Connect GitHub
# - Import Repository
# - Deploy! 🚀
```

### ⚡ **Opção 2: Via CLI (Direto)**

```bash
# Instalar Vercel CLI
npm i -g vercel

# Deploy
vercel --prod

# Seguir instruções do terminal
```

### 🌐 **Opção 3: Drag & Drop**

1. Zip a pasta do projeto
2. Acesse [vercel.com/new](https://vercel.com/new)
3. Arraste o ZIP
4. Deploy automático!
