## ✨ **Sobre o Projeto**

Um mapa interativo do Brasil onde você pode explorar as gírias de cada estado. Agora com **mais de 300+ expressões catalogadas**, desde "mano" de São Paulo até "axé" da Bahia, incluindo termos amazônicos únicos e tradições regionais autênticas. É um projeto educativo que celebra a riqueza linguística e cultural brasileira!

### 🎯 **Funcionalidades Premium**

- 🗺️ **Mapa clicável** de todos os 27 estados + DF
- 🔍 **Busca inteligente** por estado ou gíria específica  
- 📱 **Mobile-first** - Funciona perfeitamente em qualquer dispositivo
- 🎨 **Design brasileiro** com as cores da nossa bandeira
- 🧠 **Header inteligente** que se esconde automaticamente
- ➖➕ **Controles dinâmicos** para minimizar/maximizar interface
- 🏆 **Sistema de conquistas** conforme você explora
- 🐍 **Scripts Python** para análise e validação de dados
- 📊 **Analytics avançados** com insights regionais
- 🎭 **Gamificação completa** com badges e progresso

## 🚀 **Como Usar**

### ⚡ **Instalação Rápida (30 segundos)**

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/mapa-girias-brasileiras.git
cd mapa-girias-brasileiras

# 2. Servidor local (escolha uma opção)
npx serve . -p 3000              # Node.js
python -m http.server 8000       # Python
php -S localhost:8000            # PHP

# 3. Acesse no navegador
http://localhost:3000  # ou 8000
```

### 🎯 **Testando Funcionalidades**

1. **🖱️ Clique em São Paulo** → Explore "mano", "bagulho", "trampo"
2. **🔍 Digite "Bahia"** → Descubra "axé", "arretado", "mainha"
3. **📱 Teste no mobile** → Header se esconde automaticamente
4. **🏆 Visite 5 estados** → Desbloqueie "Explorador"
5. **🐍 Execute scripts Python** → Veja analytics avançados
